# PHPSupport

PHP で Package 開発を行うのに必要な実装サポートツールをまとめたものです。  
単純な実装や抽象的な機能のみを提供するツールなので、高機能なものが必要な場合は他のライブラリを利用することをお勧めします。

## Arr

配列操作をサポート

## Bootstrap

パッケージの初期処理をサポート

## Collection

コレクション実装をサポート

## Enum

列挙型の実装をサポート

## Event

同期イベントの実装をサポート

## Config

パッケージの設定をサポート

## Path

パス加工をサポート（主に結合）
